ReadMe:
You can run it directly and there is a 10 students dataset inside of the program.

      Student handle program

      1.Print the whole student set
      2.Add a new student
      3.Search the students with the firstname
      4.Search the students with the lastname
      5.Search the students with the G#
      6.Remove a student by G#
      0.Exit

Because it needs the input var which may not supported by codechef, please run it on your own IDE. There is also a codechef version which can demonstrate it automatically and feel free to contact us by unlimitediw@gwmail.gwu.edu if you need it.
